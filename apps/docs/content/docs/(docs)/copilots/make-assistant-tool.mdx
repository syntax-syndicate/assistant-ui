<Callout type="warn">
  `makeAssistantTool` is deprecated. Use a toolkit registered with
  `Tools({ toolkit })` instead.
</Callout>

The toolkit API keeps the model schema, browser executor, disabled state, and tool-call renderer in one named map:

```tsx
import { AuiProvider, Tools, type Toolkit, useAui } from "@assistant-ui/react";
import { z } from "zod";

const toolkit = {
  submitForm: {
    type: "frontend",
    parameters: z.object({
      email: z.string().email(),
      name: z.string(),
    }),
    execute: async ({ email, name }) => {
      return { success: true };
    },
  },
} satisfies Toolkit;

function FormTools({ children }: { children: React.ReactNode }) {
  const aui = useAui({
    tools: Tools({ toolkit }),
  });

  return <AuiProvider value={aui}>{children}</AuiProvider>;
}
```

See [Migrating Tools to Toolkits](/docs/migrations/toolkit-tools) for the full migration path.
