---

<Callout type="warn">
  `makeAssistantToolUI` is deprecated. Put `render` on the matching toolkit
  entry instead.
</Callout>

For a backend, MCP, or LangGraph tool, register a render-only backend entry:

```tsx
import { AuiProvider, Tools, type Toolkit, useAui } from "@assistant-ui/react";

const toolkit = {
  getWeather: {
    type: "backend",
    render: ({ args, result, status }) => {
      if (status.type === "running") return <p>Loading weather...</p>;
      return <WeatherCard location={args.location} weather={result} />;
    },
  },
} satisfies Toolkit;

function ToolUIs({ children }: { children: React.ReactNode }) {
  const aui = useAui({
    tools: Tools({ toolkit }),
  });

  return <AuiProvider value={aui}>{children}</AuiProvider>;
}
```

See [Migrating Tools to Toolkits](/docs/migrations/toolkit-tools) for examples that replace `useAssistantToolUI` and `makeAssistantToolUI`.
